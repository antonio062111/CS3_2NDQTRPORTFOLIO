Uses of Beryllium

# Use
1 Beryllium can be used in mirrors as it can be polished well.
2 Beryllium can be used in medical applications.
3 Beryllium can be used in semiconductor manufacturing.
4 Beryllium can be used in making windows for X-Ray tubes.

Fun Fact

Nicolas-Louis Vauquelin discovered beryllium in 1798 in beryl and emeralds. It only became an independent element in 1828 by Friedrich Wöhler and Antoine Bussy. Beryllium element is in period 2 and group 2 in the periodic table. It is a steel gray, strong, lightweight and brittle alkaline earth metal. Commonly found in beryl, emeralds, and minerals.
